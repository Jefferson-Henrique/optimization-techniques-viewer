ACO

With this tool you can view step-by-step how the Ant System algorithm works. Each ant has a graphic representation and will, literally, follow its chosen path.

To use this tool you can follow the basic steps:

  1. Check Configurations panel, there you can place every node where the ant can go.
  2. Click Execute on Results panel, the first algorithm iteration will begin.
  3. To advance you need to click on Deposit Pheromone then wait to ants complete this job.
  4. From now you will need to click on Next Iteration then repeat from last step.
